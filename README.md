# ColdGuard-ARM-LPC2148
IoT-enabled cold storage monitoring system built on ARM7 LPC2148. Real-time temperature &amp; humidity sensing (DHT11), door-open alerts, buzzer alarms, 16×2 LCD display, password-protected config menu, EEPROM persistence, and live ThingSpeak cloud dashboard via ESP-01 Wi-Fi. Keil µVision | Embedded C
